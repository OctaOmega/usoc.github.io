How to Install Python Virtual Environment For Windows

Python Virtual Environment For Windows

To install a Python virtual environment on Windows, you can use the built-in tool called venv. Here are step-by-step instructions:

  1. Open a Command Prompt:

    • Press Win + R to open the Run dialog.
    • Type cmd and press Enter.

  2. Navigate to your project folder:

    • Use the cd command to change to your project directory. For example: 
      cd path\to\your\project

  3. Create a virtual environment:

    • Enter the following command to create a virtual environment named "venv" (you can replace "venv" with any name you prefer): 
      python -m venv venv
    • Note: Ensure that you have Python installed, and the Python executable is in your system's PATH.

  4. Activate the virtual environment:

    • Navigate to the Scripts directory inside your virtual environment folder: 
      cd venv\Scripts
    • Activate the virtual environment by running the activate script: 
      activate
    • You should see the virtual environment's name in the command prompt, indicating that it is now active.

  5. Install packages in the virtual environment:

    • With the virtual environment activated, you can use pip to install Python packages, and they will be isolated to this environment. 
      pip install package_name

  6. Deactivate the virtual environment:

    • To deactivate the virtual environment and return to the global Python environment, simply type: 
      deactivate

USOC Endpoint Client Setup:

USOC Endpoint Client Agent will work like an API to collect Endpoint Device information such as Running Processes, Installed Applications, Network Connections, and Firewall Rules and it also provides a Remote Command Prompt to interact with the Endpoint System. Client Agent will directly communicate with Server using HTTPS request and an API (Known Vulnerability: Please check Known vulnerabilities).

There is not much installation requirement for the USOC Endpoint Client since the agent file is a compiled and built Portable Executable Binary using Flask Framework. Download the zip file from Files & Sources and follow the below installtion steps.

In Endpoints: Add System Environment Variable

  • Step 1Press Windows + R open the Windows Run prompt.
  • Step 2 Type in sysdm.cpl and click OK
  • Step 3 Open the Advanced tab and click on the Environment Variables button in the System Variable window.
  • Step 4 Select Path and Click Edit
  • Step 5 Add the the following path in the last emplty line and Click OK in all Open Windows
  • Path C:\Program Files\Usoc-Client\nssm-2.24\win64\

Download and Install the USOC Client Agent

  • Step 1Download the Usoc-Client.Zip File from Files & Sources
  • Step 2 Unzip the file "Right Click" Extract All
  • Step 3 Copy the Unzipped Folder to ` C:\Program Files `
  • Warning Do Not Rename The Folder or Files
  • Step 4 Open PowerShell as Administrator and enter Command ' cd "C:\Program Files\Usoc-Client `
  • Step 5 Run the the install.bat file by enter command ` .\install.bat `

install.bat will create a windows background service to run the flask server binary as a background service. You can interact with the sevice name: usoc_client_service For Uninstallation STOP the service and delete the foloder from C:\Program Files\. To Delete the service run `sc.exe delete usoc_client_service` from PowerShell as Administrator.
